# GameJam 抽奖软件

这是一个为Godot GameJam活动设计的抽奖软件，可以基于权重随机抽取符合条件的参赛作品。

## 功能特点

1. 基于权重的抽奖系统
2. 单个奖项不会重复获奖
3. 多个奖项不会重复抽奖
4. 支持从itch.io获取GameJam数据
5. 可配置的奖项设置
6. 华丽的抽奖视觉效果

## 使用方法

1. 启动软件
2. 点击"获取网络数据"从itch.io获取参赛作品数据，或通过"导入数据"按钮导入JSON格式的参赛作品数据
3. 添加奖项并设置数量
4. 点击"开始抽奖"按钮进行抽奖
5. 抽奖结果会显示在右侧的获奖名单中
6. 如需重新抽奖，点击"重置抽奖"按钮

## 数据格式

导入的JSON数据格式应为：

```json
[
  {
    "id": "12345",
    "title": "作品标题",
    "user": "作者名称",
    "url": "https://example.com/entry",
    "weight": 1.0
  },
  ...
]
```
